# Mahir's Eisenhower Matrix Task App

This app uses a simple GTD system to organize tasks using an Eisenhower Matrix.

## Tech stack

-   React for Frontend

## Libraries Used

-   React
-   React Router (Page routing)
-   React Hook Form (Form handling)
-   Zustand (State management)
-   HeroUI (UI library)
-   Motion
-   Lucide
-   Tailwind
